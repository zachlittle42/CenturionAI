# Future Vision — Image Generation Prompts

## How to Use These Prompts

**Individual Layer Prompts (1-10):** Each generates a single horizontal panel representing one layer of the vertical scroll experience. These are designed to be composited together into a continuous vertical panorama. Use these for focused iteration on specific sections.

**Comprehensive Prompts (A-C):** These attempt to capture the full vision in a single generation. Use these for concept art exploration, mood boards, or to brief artists/agents on the overall feel.

All prompts are optimized for high-quality image generation models (Midjourney, DALL-E 3, Stable Diffusion XL, Flux, Ideogram). Adjust aspect ratios and style suffixes as needed for your specific model.

## Production Notes

### For Midjourney
Append `--ar 16:9` (horizontal) or `--ar 9:16` (vertical) as noted. Add `--v 6.1 --style raw` for photorealistic outputs, or `--v 6.1 --stylize 200` for more artistic interpretation.

### For DALL-E 3
These prompts work well as-is. For the comprehensive prompts, you may need to simplify slightly as DALL-E 3 can struggle with extremely complex scenes. Break into sub-scenes if needed.

### For Stable Diffusion / Flux / ComfyUI
Use these prompts as the positive prompt. Add negative prompt: `"text, watermark, logo, blurry, low quality, cartoon, anime, abstract, minimalist"`. For the cross-section prompts, consider using ControlNet with a rough sketch of the layer layout.

### For Agents / Automated Pipelines
Use Comprehensive Prompt C as the master reference. Individual prompts 1-10 can be fed sequentially to generate each layer, which are then composited in order (1 at top, 10 at bottom) with overlapping blend zones.
